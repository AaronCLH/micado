Download Micado

All Micado releases are available from the pub directory. If you use Micado, we would love to hear back from you.

F#

Because Micado is developed mostly in F#, you'll need to download F# and install it on your machine in order to use Micado. Please see the INSTALL.txt file of your Micado release for the exact version of F# required for your release.
